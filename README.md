# SAT Day7 Ultimate V8 自动错题本 + 复习功能 网站包

本版本由 Day7 Ultimate V7 刷题留痕版升级而来，完整保留：

- 美式标准读音
- 单词朗读
- 例句朗读
- 朗读全部单词 / 全部例句
- 隐藏答案
- 选择题点击留痕
- 填空题输入答案
- 阅读找词题输入答案
- 本机自动保存
- 一键清除记录
- GitHub Pages 直接上传使用

V8 新增：

- 自动错题本：小测、填空、阅读找词答错后，相关词自动进入“我的错题词本”
- 薄弱词识别：错误 2 次以上会标为“重点薄弱词”
- 复习功能：可切换到“只复习错题词”模式，只显示掌握不好的词
- 错题朗读：可朗读所有错题词或错题例句
- 标为已掌握：复习后可将词移出错题本
- 清空错题本：单独清空薄弱词记录，不影响普通做题逻辑

## 上传 GitHub Pages

把本文件夹内所有文件上传到 GitHub 仓库根目录，开启 GitHub Pages 即可访问。

## 注意

所有做题记录和错题本只保存在学生当前设备、当前浏览器的 localStorage 中，不会上传到服务器。


## 本次界面优化

- 每个单词例句旁增加“暂停 / 继续 / 停止”朗读控制。
- 自我复盘小栏改为可填写、可自动填入错题词、可保存和清空的操作区。
