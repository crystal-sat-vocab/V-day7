# SAT Day7 Ultimate V7 刷题留痕版 · 美式标准读音 · 例句朗读

本网站包完整保留 Day14 Ultimate V7 刷题留痕版模板，并新增每个单词的例句朗读功能。

功能包括：

- 词表单词美式标准读音
- 每个单词配例句朗读
- 朗读全部单词、朗读全部例句、朗读阅读短文
- 选择题点击作答并保留颜色痕迹
- 填空题可输入答案并检查
- 阅读找词题可输入答案并检查
- 做题记录自动保存在本机浏览器 localStorage
- 一键清除记录，恢复全新练习状态
- 选择题库：全部、词表、小测、填空、阅读找词

## 上传 GitHub Pages

把本文件夹内所有文件上传到 GitHub 仓库根目录，开启 GitHub Pages 即可访问。
